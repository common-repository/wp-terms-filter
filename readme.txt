PLugin add filter field to admin backend taxonomy terms list to select categories in faster way.

== Description ==

This simple plugin provide great functionality for users, who have a big list of categories or custom terms in backend. 

Imagine you have about 50 categories for posts. On post creation you need manualy scroll thru all of categories to find one you need. With this plugin you will have input field where you can type category name and list of categories will be filtered.

Plugin works for default hierarchical taxonomies and for custom added hierarchical  taxonomies.



== Installation ==


1. Upload plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
